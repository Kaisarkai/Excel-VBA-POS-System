# Excel VBA Point of Sale (POS) System

A lightweight and efficient Point of Sale (Cashier) application built entirely within Microsoft Excel using **VBA (Visual Basic for Applications)** and **ActiveX Controls**.

This project demonstrates how standard spreadsheets can be transformed into interactive software tools for business automation and data management.

## ðŸš€ How to Run

Since this project contains Macros, Windows security might block the execution by default. Follow these steps to run it safely:

1.  **Clone or Download** this repository.
2.  Locate the file `PROGRAM MESIN KASIR.xlsm`.
3.  **Right-click** the file > **Properties**.
4.  At the bottom of the General tab, check **"Unblock"** (if visible) and click Apply/OK.
5.  Open the file in Microsoft Excel.
6.  Click **"Enable Content"** or **"Enable Macros"** in the yellow warning bar at the top.
